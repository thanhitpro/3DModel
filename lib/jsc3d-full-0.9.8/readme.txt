JSC3D is a model viewer that can be embeded into a web page. JSC3D is built on pure javascript and HTML5 without dependency of plugins or hardware acceleration.
JSC3D is widely compatible with major browsers that supports HTML5. A slightly modified version for InternetExplorer(version<9) is also available.

JSC3D is freely distributable under the terms of the MIT license.

The project is hosted at http://code.google.com/p/jsc3d/.
